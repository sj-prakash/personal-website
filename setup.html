My Arch Linux Setup.

My Laptop is where I do most of my work. Right now it's running Arch Linux.

I've been programming more lately so I mainly use my laptop for college, coding, watching videos and It's good enough for my needs.

The i5 in there is surprisingly still pretty capable. It takes 5 secs to open Firefox and loading any heavy site takes upwards of 3 seconds.

For my terminal I use iTerm2 and Oh My Zsh to customize it. The color theme I'm using is Nordic. For audio spectrum visaulizer I'm using Glava
